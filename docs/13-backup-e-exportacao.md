# Backup e Exportação

## Banco

Arquivo:

storage/database/biblioteca.db

## Backup

Gerar cópia:

storage/backups/

Formato:

biblioteca_YYYYMMDD.db

## Exportações

storage/exports/

Arquivos:

- acervo.xlsx
- usuarios.xlsx
- emprestimos.xlsx

- acervo.csv
- usuarios.csv
- emprestimos.csv

## Objetivo

Permitir sincronização via:

- Google Drive
- OneDrive
- Pasta compartilhada
