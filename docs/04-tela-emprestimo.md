# Tela Empréstimo

## Busca de Usuário

Campo principal:

Nome Completo

Busca dinâmica.

## Dados exibidos

* Nome Completo
* CPF
* Data Nascimento
* Celular
* Email
* Número Cadastro
* Quantidade Empréstimos Ativos

## Botão

Realizar Empréstimo

## Modal de Acervo

Busca por:

* Número do Exemplar
* Título

Resultado:

* Número Exemplar
* Título
* Autor
* Classificação
* Status

Somente itens disponíveis podem ser selecionados.

## Confirmação

Exibir:

* Usuário
* Exemplar
* Título
* Data Empréstimo
* Data Prevista Devolução

Botão:

Confirmar Empréstimo
